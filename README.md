Advanced Form Handling & Validation

## 🚀 Tech Used
- React + Vite
- react-hook-form
- Yup + @hookform/resolvers

## 📋 Features
- Full registration form with 8 fields
- Client-side validation using Yup schema
- Real-time error messages on submit
- Password strength & match validation
- Success screen after valid submission

## ▶️ Run Locally
npm install
npm run dev

## 💡 Key Concepts
- `useForm()` manages form state efficiently
- `yupResolver()` connects Yup schema to react-hook-form
- `formState.errors` displays field-level error messages
